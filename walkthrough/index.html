What this demo contains

This is a deliberately simple app (task board, ~710 lines, zero dependencies). The point is not the app's complexity — it's the methodology's output. SDD scales the same way for apps 100x this size.

LayerContent
project-profile.yamlSDD configuration — defines project goals, stack, maturity. Used by sdd init to activate the right programs.
Live AppFunctional Kanban board — drag tasks, keyboard nav, localStorage persistence
Inception Artifacts12 docs: requirements, business rules, verification Q&A, personas, stories, HLD, components, dependencies, plans
Construction Artifacts9 docs: per-unit functional design + NFR requirements + code summary
Operations Artifacts3 docs: user guide, developer guide, architecture
Tests25 unit tests + 4 property-based tests (organized by requirement ID)
Audit TrailTimestamped gate approvals and phase transitions with evidence
